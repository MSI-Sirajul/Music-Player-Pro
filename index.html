Key Features

⚡

Native Performance

Built without AndroidX or external libraries. Pure Java and XML for ultra-fast, lightweight performance and low RAM usage.

🎨

Material Design 3

Fully custom XML drawables replicating the modern Material You aesthetic. Dynamic themes (Light/Dark mode) support.

📂

Folder Browsing

Advanced file management logic allows you to browse and play music directly from your device folders.

🔒

Secure & Stable

Targeting Android 14. Implements Foreground Services, Audio Focus handling, and Wake Locks for uninterrupted playback.

🎧

Pro Controls

Lock screen controls, Headphone unplug auto-pause, Gapless playback logic, and a global Mini Player.

❤️

Favorites & Queue

SQLite database integration to save your favorite tracks and a dynamic playing queue management system.
