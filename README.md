##  Key Features

*   **⚡ Native Performance:** Built without AndroidX/Jetpack libraries for minimal RAM usage.
*   **🎨 Material 3 Design:** Custom XML drawables creating a modern, glass-morphism aesthetic.
*   **📂 Folder Browsing:** Navigate and play music directly from your device directories.
*   **🌗 Dynamic Theme:** Full support for Light and Dark modes.
*   **🎧 Pro Controls:** Lock screen controls, Headphone auto-pause, and Notification actions.
*   **❤️ Favorites & Queue:** Integrated SQLite database for favorites and dynamic queue management.
*   **🔍 Smart Search:** Filter tracks and folders instantly.

##  Technical Specs

*   **Language:** Java
*   **Min SDK:** Android 5.1 (Lolipop)
*   **Target SDK:** Android 14 (Upside Down Cake)
*   **Architecture:** Service-based (Foreground Media Service)
